# Vegetation Management qualification

Run `cargo run -p vegetation-management --example workflow_qualification --release` for the CI-safe geometry/workflow qualification. The benchmark measures only deterministic polygon validation overhead. Scenario tests separately exercise eight independent safety-inhibit inputs, revision and authority gates, resource snapshots, simulated evidence, and longitudinal assessment.

This evidence does not establish ecological effectiveness, field robot productivity, detection performance, or safe operation in an undeclared ODD. Completion evidence is intentionally separate from longitudinal effectiveness evidence.
