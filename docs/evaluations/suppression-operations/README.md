# Suppression actuation qualification

The enabled modes are deterministic simulation and human-supervised HITL teleoperation. No production autonomy, target selection, automatic re-arm, or envelope widening exists. Scenario tests cover the independent inhibit and occurrence path for intrusion, uncertainty, spatial/environmental/dose/rate/pressure/tool/sensor/actuator/supervision/stop failures.

The release harness is an in-process timing sanity check only. It is not evidence of physical valve closure, residual-flow performance, communications latency, agent efficacy, or field safety; those require HITL and controlled-field evidence under the promoted ODD.
